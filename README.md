# ร้านของฉัน - อาหารตามสั่ง

ยินดีต้อนรับสู่เว็บไซต์ของร้านอาหารตามสั่ง "ร้านของฉัน" ซึ่งเป็นเว็บไซต์ที่ออกแบบมาอย่างดีเพื่อแสดงถึงร้านอาหารตามสั่งของคุณอย่างมืออาชีพ

## โครงสร้างเว็บไซต์

- `index.html` - หน้าหลักของเว็บไซต์
- `pages/` - โฟลเดอร์เก็บหน้าเว็บไซต์ย่อย
  - `about.html` - หน้าเกี่ยวกับร้าน
  - `menu.html` - หน้าเมนูอาหาร
  - `contact.html` - หน้าติดต่อ
  - `promotion.html` - หน้าโปรโมชั่น
  - `review.html` - หน้ารีวิว
- `css/` - โฟลเดอร์เก็บไฟล์รูปแบบ
  - `style.css` - ไฟล์รูปแบบหลัก
- `js/` - โฟลเดอร์เก็บไฟล์จาวาสคริปต์
  - `script.js` - ไฟล์จาวาสคริปต์หลัก
- `img/` - โฟลเดอร์เก็บรูปภาพ

## คุณลักษณะ

- หน้าเว็บแบบ Responsive รองรับทั้งบนมือถือและเดสก์ท็อป
- โหมดกลางวันและโหมดมืด
- รูปแบบที่สวยงามและทันสมัย
- หน้าเว็บครบทุกประเภทที่ร้านอาหารตามสั่งต้องการ
- โค้ดที่สามารถบำรุงรักษาได้ง่ายและสามารถปรับแต่งได้
- ภาพที่มีเอฟเฟกต์ hover ที่นูนขึ้นและมีเงาเมื่อเมาส์ชี้

## การใช้งาน

1. เปิดไฟล์ `index.html` ในเบราว์เซอร์ของคุณเพื่อดูหน้าหลัก
2. ใช้ลิงก์นำทางเพื่อเข้าชมหน้าอื่น ๆ
3. แก้ไขไฟล์ HTML เพื่อปรับเนื้อหาให้เหมาะสมกับร้านของคุณ
4. แก้ไขไฟล์ CSS และ JavaScript เพื่อปรับแต่งลักษณะการใช้งาน

## การปรับแต่ง

- แก้ไขข้อมูลร้านอาหารในไฟล์ `index.html`, `about.html`, และ `contact.html`
- เพิ่มเมนูอาหารในไฟล์ `menu.html`
- ปรับปรุงรูปลักษณ์โดยแก้ไข `css/style.css`
- เพิ่มรูปภาพของคุณในโฟลเดอร์ `img/` และแทนที่ลิงก์รูปภาพในไฟล์ HTML
